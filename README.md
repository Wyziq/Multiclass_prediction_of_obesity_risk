## Описание проекта
### Проблема
Рост заболеваемости ожирением во всем мире представляет серьезную угрозу для общественного здоровья. Отсутствие эффективных инструментов для раннего выявления групп риска приводит к поздней диагностике, развитию сопутствующих заболеваний (диабет,сердечно-сосудистые патологии) и увеличению нагрузки на систему здравоохранения. Существующие методы оценки риска часто основаны на ограниченном наборе параметров и не учитывают комплексное влияние образа жизни, пищевых привычек и генетических факторов.

### Цель проекта
Разработать модель машинного обучения для многоклассовой классификации уровня риска ожирения на основе комплексного анализа антропометрических, поведенческих и диетологических данных. Создать инструмент для раннего выявления пациентов с различными степенями риска ожирения для своевременного профилактического вмешательства.

### Ожидаемые результаты
- Обученная и валидированная модель классификации с точностью не менее 85%
- Аналитический отчет о значимости различных факторов риска
- Документация по использованию модели и интерпретации результатов
- Рекомендации по профилактическим мероприятиям для каждой группы риск
